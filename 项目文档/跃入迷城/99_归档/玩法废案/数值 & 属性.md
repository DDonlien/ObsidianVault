**属性构成**

**自驱动属性**

- 属性本身提供战斗价值，属性可以直接被提升

**驱动属性**

- 属性本身不提供战斗价值，属性可以直接被提升，驱动其他有价值属性提升

**被驱动属性**

- 属性本身提供战斗价值，不可以直接被升级，被其他有价值属性提升

**非数值属性**

- 技能槽

**非战斗属性**

- 没有直接战斗价值的属性，或不直接在战斗中升级关联的属性

**伤害类型**

- 游戏中所有伤害附带伤害属性，存在以下几种**伤害属性**：

**v0.1.0**

动能 (Kinetic)：对护甲造成50%伤害，对生命造成150%伤害，优先伤害护甲

电磁 (Electrical)：对护甲造成150%伤害，对生命造成50%伤害，优先伤害护甲

热能 (Thermal)：对护甲造成100%伤害，对体力造成100%伤害，优先伤害护甲

**v1.0.0**

辐射 (Radioactive)：无法对护甲造成伤害，越过护甲直接对体力造成100%伤害

神经 (Neural)：对护甲和体力造成50%伤害，同时伤害两者

**武器补正**

调整后的武器补正如下：

1. 补正随等级成长，从0级的0.5~1成长到5级的1.5~2
2. 补正有偏差矫正，奖励偏科生（因为义体的属性在奖励全科生，从而平衡）
3. 极端情况下，全偏科武器+全偏科义体，满级DPS是全平衡武器+全平衡义体的2倍，刚好，分别是
1. 偏科：580
2. 全科：280
3. 大致是全偏科毕业可以10秒平A击杀第一个BOSS（5000血）的水平，刚好
4. 此时矫正前伤害大致也是这个数，差异会更小一些（525:300）刚好对应上全科武器可以用更多类型的技能，而偏科可以在更少的类型中获得更大的伤害收益，注意，偏科很可能是玻璃大炮
5. 为了在不降低初始武器伤害的前提下实现该调整，降低了武器的基础伤害成长，从1级+10改为了1级+5




武器总伤害（dps）=武器面板/攻击间隔

武器面板=基础攻击力+∑（基础攻击力\*属性补正系数\*属性补正效率）  

补正系数：指某项属性达到99时，所能提供的补正攻击力和基础攻击力的比值。

属性补正系数（不包含下限）：

- S ＞ 200%
- A = 150%~200%
- B = 100%~150%
- C = 50%~100%
- D = 25%~50%
- E = 0%~50%
- - = 0

属性补正效率函数：

y(x)=0.02x,if 0≤x≤20,

y(x)=0.01x+0.2,if 20<x≤50,

y(x)=0.00833x+0.28333,if 50<x≤80,

y(x)=0.00263x+0.73684,if 80<x≤99,

y(x)=1,if x=100

@Decay

对于霰弹枪等一动多发的武器（调用1次GA，生成多个伤害体），每个子弹均分总伤害，武器面板还是正常显示总伤害，即显示“校正后伤害 x 发射子弹 x 每动判定”

**子弹散布**

子弹散布由以下四个参数构成，同步物品列表的武器列表sheet[物品列表](https://dikpa4hrtn9.feishu.cn/wiki/IbdswSAf1iFX9Ik5LRDcg02LnTb?from=from_copylink)

- 当前散布张角：单位为deg，表示武器子弹当前可散布到的最大角度
- 最大散布张角：单位为deg，表示武器当前散布张角的最大值
- 扩张速度：输入参数为X%，输出单位为Ydeg，Y = X%(最大散布张角)；每次攻击，当前散布张角会增加Ydeg
- 回馈速度：输入参数为X%，输出单位为Ydeg，Y = X%(最大散布张角)；每秒，当前散布张角会减少Ydeg

子弹在当前散布张角内，以鼠标指向为中心正态散布

如果子弹散布由于外界因素变化，需要详细描述变化对象

近战武器没有子弹散布

散弹枪类武器用“当前散布”实现散弹效果，**hardcode为平均而非正态分布**@Decay
