
**角色系统 & Build**

游戏角色（无论敌我）由以下几个子系统（或者其中的一部分）构成

详细的属性记录在此表格：[角色列表](../../../跃入迷城 - 数据库/角色列表.md)

**属性系统**

- 玩家和敌人均有以下几个显性的资源值
- 生命：生命值
- 体力：大部分动作（翻滚、冲刺、攻击、使用技能）消耗体力；体力上限等于剩余生命值；停止消耗体力的动作后，体力会自动恢复
- 能量：使用技能一般会消耗能量；消耗体力会恢复能量
- 护甲：护甲值
- 除了消耗的资源值，玩家还有其他几个属性，详见腾讯表格
- 属性有显性和隐性，可以直接增减and/or跟随其他属性按比例增减
- 敌人不用（可以，但不用）复杂的属性系统驱动，直接配置必须的数值即可 
- 例如：玩家有基础体力，且基于耐力和力量获得外体力；但敌人直接配置目标数值的基础体力极客
- 所有数值（资源、属性）都由以下结构组成
- 实际值 = 基础值 + 动态增长 + 固定增长 
- 基础值：一开始就配好了
- 动态增长：基于公式，随其他属性变动 
- 如：生命动态增长值 = 力量 x 2.5 + 耐力 x 2.5
- 固定增长：额外附加值 
- 如：“获得额外50生命值”
- 敌人用固定增长就行
- 当玩家获得“提升20%生命值”之类的比例强化时，其实是在“固定增长”里增加一个20%的比例系数，也就是说固定增长不一定是个常数

**义体系统**

- 玩家可以装备8种（暂定）不同的义体，每种不重复
- 类似于常规RPG游戏的装备
- 8种分别对应以下位置： 
1. 眼球
2. 皮肤
3. 手臂
4. 腿部
5. 骨骼
6. 心脏
7. 呼吸系统
8. 神经系统
- 装备义体消耗玩家的义体容量
- 类似于常规RPG游戏的负重
- 当义体容量被消耗超过50%、75%、100%，玩家会受到负面效果
- 义体容量在局外（公寓）进行升级
- 义体包含构成属性，详见[物品列表](../../../跃入迷城 - 数据库/物品列表.md)；其中主要的几个属性如下：
- 品质：决定其他属性；通过局内升级而提升
- 属性：为玩家提供基础属性；随品质提升而提升
- 义体技能槽：该武器能装备的武器技能上限取决于技能槽的数量；随品质提升而提升
- 义体附带义体技能，且可以进行融合打印
- 义体技能可能是主动或被动 
- 被动技能持续生效
- 主动技能会添加到玩家的技能目录，玩家需要将主动技能配置到施法栏 
- 玩家用Q/E释放技能，一次性只能调用1组技能栏（2个技能），用1、2、3、4等横排数字键切换技能组，技能组数量作为局外升级项目 
- 手柄情况下，左右方向键切换技能组，上下切换武器；鼠标键盘情况下，滚轮切换武器
- 切换技能组后，技能会先CD；局外升级可以减少切换的第一次CD
- 来源&掉落算法：义体打印室

**武器系统**

- 玩家可以任意持有3~6把、使用1~2把武器
- 持有数量通过局外升级
- 使用数量取决于武器本身是否可以双持
- 武器可以从以下2个维度分类
- 攻击方式：近战武器、远程武器、投掷武器、部署武器
- 伤害主属性：动能、电磁、热能 
- 不同的伤害类型对护甲、生命造成不同比例的伤害，且有不同的伤害优先级 
- 动能 (Kinetic)：对护甲造成50%伤害，对生命造成150%伤害，优先伤害护甲
- 电磁 (Electrical)：对护甲造成150%伤害，对生命造成50%伤害，优先伤害护甲
- 热能 (Thermal)：对护甲造成100%伤害，对生命造成100%伤害，优先伤害护甲
- 大部分武器攻击消耗弹药，弹药基于伤害主属性分类
- 武器可能造成非主属性的伤害，此类伤害不影响弹药适配

```<p>- 弹药获取的途径： </p><p>- **方案1：每种弹药有有限的备弹；**</p><p>- 每次换弹消耗固定数量（单价数量）的备弹；从敌人尸体拾取，或从补给房间获取</p><p>- Pros：最真实；可以通过子弹掉率配置强度</p><p>- Cons：爽感最低；玩家需要频繁扫尸体</p><p>- **方案2：每种弹药消耗一定的「备弹值」；**</p><p>- 每次换弹消耗一定比例的「备弹值」；通过随身携带的弹药3D打印机自动恢复，通过技能、升级，提升恢复速度</p><p>- Pros：爽感和真实性平衡；追加一个升级点提升游戏build深度</p><p>- Cons：平衡性设计变得更复杂</p><p>- **方案3：每种弹药有无限的备弹；**</p><p>- Pros：最爽快</p><p>- Cons：真实感降低；和“资源管理”的魂系战斗主题违背</p>```
- 武器包含构成属性，详见[物品列表](../../../跃入迷城 - 数据库/物品列表.md)；其中主要的几个属性如下：
- 品质：决定其他属性；通过局内升级而提升
- 基础伤害：基础的伤害值；随品质提升而提升
- 伤害补正：基于玩家属性获得伤害补正；随品质提升而提升 
- 武器补正的E → A代表了不同的补正系数，补正伤害 = 属性 x 补正系数 
- / = 0
- 0< E ≤ 0.2
- 0.2 < D ≤ 0.4
- 0.4 < C ≤ 0.6
- 0.6 < B ≤ 0.8
- 0.8 < A ≤ 1
- 1 < S
- 武器技能槽
- 该武器能装备的武器技能上限取决于技能槽的数量；随品质提升而提升
- I (1) = 1
- II (2) = 2
- III (3) = 3
- IV (4) = 4
- V (5) = 5
- 武器附带战技和武器技能，且可以进行融合打印
- 战技通过鼠标右键释放，每把武器只有一个 
- 战技可能有主动and/or被动效果
- 使用战技可能消耗体力/能量/生命/护甲
- 武器技能只有被动，仅对使用中的该武器生效
- 子弹散布@Decay@Cade
- 当前子弹的projectile不基于力，而是基于给定方向初速度实现发射
- 子弹散布会以“在生成子弹前，预先调整初速度方向”实现偏移，每次发射时，给子弹一个散布偏角
- 子弹散布涉及以下4个变量：
- 最大散布角度：无论连续射击多久，子弹会散布到的最大张角，恒定
- 当前散布角度：每次射击后变大，随时间减小的散布角度；不会大于最大散布角度；默认=0，不小于0
- 扩张速度：发动攻击后，当前散布角度扩大的量；单位deg（角度，如果是弧度，请转换）
- 回馈速度：没有在攻击时，当前散布角度变小的速率；单位deg/s（角度，如果是弧度，请转换）
- 子弹散布遵循以下规则：
- 每次射击时，子弹会在当前散布角度内散布
- 由于当前散布角度不会<0，且默认为0因此第一发子弹会有精准的落点
- 该散布符合正态分布，也即越靠近瞄准位置，落点出现的概率越高
- 停止射击时，当前散布角度逐渐变小回0
- 来源&掉落算法：武器打印室

**Build设计**

**动能 - 暴击流**

武器可以造成暴击

暴击带来临时增益状态

换弹、减少攻击频率会增加暴击率、暴击伤害

**热能 - 灼烧流**

累积一定热能伤害，会对敌人施加“灼烧”debuff

通过切换攻击目标，可以快速扩散debuff

**电磁 - 扩散流**

伤害可以扩散到附近其他敌人

受到过电磁伤害的敌人受“蓄电”debuf到更高电磁伤害

**纪念碑 - 伤害交替流**

通过施加一个种类的伤害，强化另一种类造成的伤害

**生物科技 - 卖血流**

受到伤害时增加造成的伤害

低体力时获得强化（尤其是体力恢复）

**东海工业 - 护甲流**

减少体力上限，获得大量护甲

完整build情况下，体力归0也不会死亡

**律动智能 - 技能流**

增加直接技能造成的伤害，减少技能冷却

增加切换技能后技能造成的伤害

**军用科技 - 武器大师**

切换武器种类造成额外伤害

大幅提升战术武器的效果

**野望 - 隐身近战流**

近战武器伤害大幅提升

增加潜行类技能的效果，以及前行状态下的伤害
