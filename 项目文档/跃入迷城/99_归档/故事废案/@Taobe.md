**[[莫里](../设定Wiki/角色/友方NPC/“掮客” - 莫里.md)]**

```**设计：OK指引：** 用于快速构建角色核心，并通过叙事碎片呈现给玩家。```
- 参考：《黑客帝国2》法国人

**第一部分：核心概念**

- **代号 / 姓名:**
- 莫里 (Morie)，但他在公司客户面前会用一个更正式的名字。
- **身份 / 阵营:**
- 表面是**中立公司**的高级顾问/情报分析师，暗中是独立的情报贩子**私家侦探**，一个在刀尖上平衡各方利益的利己主义者。
- **核心驱动力:**
- 在确保自身绝对安全的前提下，最大化个人利益，并最终脱离不夜城这个泥潭。
- **功能性质:**
- [✓] **街角观察者:** 在前者基础上，其本体会出现在安全区或已清场的战斗区域。可当面对话、提供情报、任务或交易。可被玩家攻击/击杀，但不会主动介入战斗。*(他会偶尔出现在安全屋，或者通过加密通讯联系玩家，用情报换取玩家收集到的“黑匣子”)*

**第二部分：角色深度**

- **个性 / 行为风格:**
- 优雅、伪善、言辞考究且充满暗示。他从不直接表露自己的意图，善于用利益和风险来引导他人。对玩家看似友善，实则视其为一枚有用的棋子。
- **与玩家的关系:**
- 情报贩子 → 危险的合作者 → 背叛者。

**第三部分：叙事弧光**

- **关键过往:**
- 曾与\*\*冈特(Ghent)\*\*是生意伙伴/朋友？搭档，但目睹了太多理想主义者的失败后，他彻底抛弃了所谓的“正义”，转而信奉“只有活下来并且站在高处，才能改变一切”的信条，并为此不惜一切代价。
- **初次邂逅:**
- 在玩家完成冈特的几个任务后，莫里会主动联系玩家，声称冈特的做法太天真，并愿意用更高的价格收购玩家手中的情报（黑匣子），暗示他有更高效的“处理方式”。
- **最终宿命:**
- 坏结局：得罪大人物，在冈特因错误情报而被害后，莫里会最后一次联系玩家，表达一番惋B样的惋惜，然后彻底消失，动用所有资源抹除自己存在的痕迹，成功“出走”不夜城。
- 好结局：冈特死了，虽然惋惜，但他继续干这个。
- 和冈特争取玩家的资源：谁的资源越多结局越好。
- **对结局的影响:**
- *[✓] **部分影响 (Enrichment):** 角色的故事线会“润色”主线结局。不会增加新的结局选项，但会改变结局动画的细节、后日谈的描述、或最终战役的参与人员等。*
- *(他的存在与否，决定了冈特支线的最终结局，并会在结局的旁白中增加一句“有人在混乱中悄然离场，带走了不为人知的秘密”，丰富了世界的复杂性。)*

` `**[[革命家](../剧情思考/革命家.md)]**

```**设计指引：** 用于快速构建角色核心，并通过叙事碎片呈现给玩家。```
**第一部分：核心概念**

- **代号 / 姓名:** 革命家，刘世平
- **身份 / 阵营:** 黑客组织（革命军）首领，革命军的目标是摧毁脑租赁行业。
- **核心驱动力:** 摧毁脑租赁行业，打破权贵阶级的脑算力资源垄断，让所有人都能公平地使用自己的脑算力。
- **功能性质:**
- [✓] **临时盟友 (Temporary Ally):** 在**街角观察者**的基础上，在特定区域、任务或条件下，会受邀与玩家并肩作战。合作关系可能是脆弱和短暂的，为玩家提供暂时性的战力。*(参考：《艾尔登法环》中可召唤参与BOSS战的NPC)*

**第二部分：角色深度**

- **个性 / 行为风格:** 前期装出来一副态度轻浮，毛手毛脚的滑稽小混混样子（参考火影阿飞or韦小宝）；后期展露出真实性格，坚定，隐忍，为达目的不择手段（参考久世英雄or卫宫切嗣）。另外强调一下“不择手段”的程度，他可以为了达成自己的目标牺牲一切，比如自己的生命，建立的组织，栽培的手下，亲密的战友，等等。
- **与玩家的关系:** 协助者*→战友or敌人（根据玩家的选择）*

**第三部分：叙事弧光**

- **关键过往:** 出身家庭良好的大学教师，遇见了太多因为家庭条件不好，只能努力学习让自己脑算力变多然后出租脑算力的穷孩子，也遇见了太多因为家庭条件优越，整日游手好闲却可以继承庞大脑算力的公子哥，觉得自己前途无望就自暴自弃的，不想继承家业但也只能被迫同意的，等等。
- **初次邂逅:** 假扮成黑客组织一名菜鸟外勤的刘，遇到了同样来某一章地图调查的玩家。刘表明了自己是黑客组织一员的身份，觉得双方目的一致可以建立合作关系，并以某个道具为筹码，希望玩家能送他一程。
- **最终宿命:** （以下提到的死亡都是永久死亡，不能复活的那种）
- 如果玩家完成了刘的全部任务，刘会牺牲自己找到能摧毁脑租赁行业的关键道具并交给玩家，委托玩家帮他继续完成使命。（完成这条任务线会给玩家解锁一个全新的可选结局）
- 如果玩家只完成了刘的部分任务，并最终选择成为一名vip，此时刘会作为隐藏boss出来狩猎玩家并中断刘的任务线。（这条任务线会让刘成为玩家通向vip结局前的隐藏boss）
- **对结局的影响:** 
- 新增结局：玩家继承了刘的遗愿，以自己也永久死亡的代价摧毁（or改善）了脑租赁行业。如果玩家与刘建立了亲密关系，则刘会在死前额外牺牲一些东西留一个后手保住玩家的命。
- 可能存在的额外机制：玩家可以在刘牺牲前，通过完成一系列复杂的隐藏任务拿到一个既能保住刘的命也能获得革命关键道具的额外机制。并且如果玩家与刘建立了亲密关系，则刘可以在结局处保住玩家的命，达成双活的革命结局。
