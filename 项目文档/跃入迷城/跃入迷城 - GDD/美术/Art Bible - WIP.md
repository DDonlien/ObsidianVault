**尺寸**

1. 画布 = 3000x4000，dpi = 300
2. 4000px的纵向像素对应4000cm的高度，绘制时反应正确高度
3. 标准头身比7.5，如果需要表现大头、小头，酌情调整
1. 180cm身高7.5头身比
2. 140cm身高7.0头身比，避免大头娃娃
3. 220cm身高8.0头身比，避免超小头双开门
4. 具体的公式如下：头高 = 身高/(0.0125\*身高+5.25)

**制作步骤**

我们通过多个步骤完成绘制，确保风格一致，并且验收内容可控

\1. **动作线稿**

```<p>1. **定义：**最简单的素描小人插图，明确角色的动作，嘴、眼位置，以及大型的物件在画面中的位置</p><p>2. **技法：**在该步骤**过程中**，**可以**用模型辅助工具明确角色动作，确保动作合理，光照定义清晰，镜头角度明确。建议的工具如下：</p><p>a. https://webapp.magicposer.com/</p><p></p>|![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.002.jpeg)``` :-: |
\2. **形象线稿**

```<p>1. **定义：**包含角色的五官、衣物细节、手势、发型等在内的素描线稿。</p><p>2. **技法：**在该步骤**开始前**，**应该**用模型辅助工具明确角色动作，确保动作合理，光照定义清晰，镜头角度明确，**并保留模型截图**。建议的工具如下：</p><p>a. https://webapp.magicposer.com/</p><p></p>|![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.003.png)``` :-: |
\3. **色块**

```<p>1. **定义：**包含角色肤色、物件颜色在内的色块稿，勾勒出身体上的色彩分形</p><p>2. **定义：**在3.1的基础上，增加黑色的阴影块，用光影明确面与面的基础关系</p><p>3. **技法：**套索工具勾勒出角色外形和大面积色块，整体线条多由大量直线和少量曲线组成。所有被藏在主体身后的对象，通过间隙分割开，体现出层次感（装备、头发、物品）</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.004.jpeg)</p><p>3.1的目标成果</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.005.png)</p><p>3.2的目标成果</p>``` :-: | :-: |
\4. **基础光影**

```<p>1. **定义：**在色块上增加非黑阴影的二分色块</p><p>2. **定义：**在4.1的基础上，增加主光源带来的高光，以及从黑阴影过渡出的淡阴影</p><p>3. **定义：**在4.2的基础上，用涂抹增加肌肉/起伏细节带来的更淡的阴影</p><p>4. **技法：**整体控制在3~5个色阶</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.006.png)</p><p>4.1</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.007.png)</p><p>4.2</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.007.png)</p><p>4.3</p>``` :-: | :-: | :-: |
\5. **强化光影**

```<p>1. **定义：**加强人物身上（物品上）自然的光影对比</p><p>2. **技法：**通过**高光（1）**位置和**背光（2）**位置2种**单独的光色**（与肤色不同）叠加图层来实现。</p><p>a. **高光（1）**位置和实色接近</p><p>b. **背光（2）**（黑色往平光过度）的位置用黑色往实色过度，或者增加对比色系，看实际情况。</p><p></p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.008.png)</p><p>5.1，图上的纹理不应该出现</p>``` :-: |
\6. **增加轮廓光**

```<p>1. **定义：**在黑色阴影位置，增加5.1中身体**轮廓光**</p><p>2. **技法：**</p><p>a. 用叠加图层或者柔光图层，把**轮廓光**特殊色作为\*藏色\*加入画面</p><p>b. 颜色与主题色做对比</p><p>c. 轮廓光饱和度和不透明度只能高一个，并且要平衡整体画面颜色（如果人已经很花，那这里就简单点）</p><p>d. 如果环境光颜色变化，整体遮罩正片叠底处理</p><p></p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.009.png)</p><p>6.1，图上纹理以及圈出的对象不应该出现</p>``` :-: |
\7. **增加纹理**

1. **定义：**在高光向平光过度的位置，增加与底色融合的波点纹理，体现整体艺术风格
2. **技法：**用取样笔刷5 + 棋盘格纹理 + overlay → 实现半透明的texture

\8. **增加\*风味\*** 

```<p>1. **定义：**增加装饰性的、带纹理的异色点缀，平衡画面饱满度（哪里东西多，哪里少，少的加）</p><p>2. **技法：**颜色使用背光的特殊色，以及其近似色</p><p></p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.010.png)</p><p>8.1，需要增加的画面</p>``` :-: |
\9. **增加色差**

```<p>1. **定义：**增加反色的色差轮廓，垫在阴影侧位置，提升整体画面的\*赛博朋克感\* ，**完稿**</p><p>2. **技法：**复制先前所有图层合并，使用色相偏移/色差功能，关闭red channel</p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.008.png)</p><p>9.1，对话插图完稿</p>``` :-: |
\10. **完成概念稿**

```<p>1. **定义：**以模型创作为目标，补充细节设定，完成概念稿</p><p></p>|<p>![...](../../跃入迷城%20-%20GDD/美术/images/Art%20Bible%20-%20WIP.011.png)</p><p>10.1 概念稿完稿</p>``` :-: |

1. **定义：**重新循环步骤1~9，完成同一角色的其他动作，表达不同情绪或人设状态；如有必要，补充新的概念高（10）
