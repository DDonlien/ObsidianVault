**游戏流程**

**推进流程**

1. 离开电梯间，进入战斗
2. 关卡生成式就会确定有多少敌方单位，会分布在： 
1. 直接生成在场地内的敌人、窗外的支援飞车 → v0.0.1
2. 房间动态变动的产生的敌人 → TBD
3. 在战斗过程中，搜刮补给房间获得奖励
4. 击杀所有敌人，拾取电梯权限 
1. 故事设定：电梯权限只有一份，在同楼层敌人之间基于局域SDN动态转移，所以只有最后击杀的敌人有权限，cc@Serein@Jade
5. 回到电梯间，选择要前往的楼层 
- 电梯可以让玩家前往其他关卡，权限决定可选范围 
- 视觉：@Arthur Li@Ke Huang
- ~~拿着电梯权限回到电梯间的玩家，会看到某几个电梯的全息显示变为：“ACCESS GRANTED - To LvX”，没有权限的电梯还是“NO ACCESS”；~~
- 直接在固定的一个电梯上提示交互，交互后二级菜单选择要前往的楼层
- 同一个位置的电梯，只移动固定楼层：比如最右上角的固定是+1层，以此类推

```<p>- **权限方案1：电梯可以向上或向下坐（+1、+2、+3、-1、-2、-3）**</p><p>- 如果向下的楼层有已经去过的楼层，则不能访问更低的楼层 </p><p>- 如玩家从Lv3→Lv5，在Lv5拿到Access=3，则最多向下回到Lv4，向上到达Lv8</p><p>- 继续上述假设，如果玩家在Lv8拿到了Access=3，向下回到Lv6（显然，不能回到Lv5）则Lv6不论掉落Access=几，+2都会NO ACCESS（不能回到Lv8）且此时Lv7至少会掉落Access=2。</p><p>- Pros：关卡设计更新颖；在保证期望关卡数不变的前提下，需要设计的关卡更少；</p><p>- Cons：关卡流转机制开发上比较复杂，玩家体验不直观；</p><p>- **权限方案2：电梯只能向上坐（+1、+2、+3）**</p><p>- 如果可访问的高层有Boss关，最多只能到Boss关</p><p>- Pros：关卡流程更简单直接</p><p>- Cons：需要制作更多关卡保持期望关卡数不变；玩家会快速跳过非Boss关；反之，玩家可能在非Boss关耗时太久，变的太强；</p>```
6. 进入电梯并load下一关

```<p>**逃生逻辑**</p><p>1. 玩家可以在有开放阳台（边缘）的位置叫逃生飞车，逃生飞车会一段时间后前来结束这次任务，呼叫必须在楼层封闭（清完敌人）前，并且： </p><p>a. 方案1：如果等待期间清完了敌人，呼叫也会停止</p><p>b. 方案2：如果等待期间清完了敌人，呼叫不会停止</p>```

**战争迷雾**

**方案1：没有战争迷雾**

**Pros：**最经典的俯视角视野

**Cons：**

- 玩家视野和角色视野不对应，无法设计转交杀之类的结构
- 缺少新意和卖点

**方案2：锥形战争迷雾**

```<p>- **整体诉求**</p><p>- 玩家的在屏幕上会看到以主控角色为中心，朝向主控角色面前扇形展开的范围内的视野</p><p>- 锥形范围以外的空间，应当由2层战争迷雾遮挡</p><p>- 第一层为黑色，在完全未探索（玩家视野没有扫过）的区域覆盖</p><p>- 第二层为灰色，在玩家已探索过，但现在未在视野内的区域覆盖</p><p>- 基于以上设计，玩家屏幕内的空间应当被分为以下3个区域：</p><p>- 黑色迷雾区域</p><p>- 灰色迷雾区域</p><p>- 视野区域</p><p>- 灰色和黑色区域应该都是黑色遮罩+不透明度实现，灰色不透明度 = {50%}</p><p>- 灰色迷雾区域内，所有“character”不可见</p><p>- 灰色迷雾区域内，所有“actor”维持最后一次看到的状态（不刷新，war3）</p><p>- 视野区域的扇形，其锥顶应当以玩家主控角色为中心，扩展一个半径为Y的圆（周边视野）</p><p>
同步自文档: <https://dikpa4hrtn9.feishu.cn/docx/TSiXdxrPLoYD22xtLracy24Xn9e#STa6dIqK0sNGWybayDzcib27n5b></p>```

**局内奖励**

- 每一层战斗关卡中，会有0~N个奖励房间 
- 并非所有房间都是奖励房间，有可能打开门里面是埋伏或剧情
- 玩家在乘坐电梯前**无法**看到对应楼层奖励类型
- 玩家必须在完整清空楼层前获得奖励，否则奖励会被锁定
- 玩家在局内会遇到以下奖励房间 
- 设定上，没有特殊解释的奖励房间都是给公司安保人员（敌人们）的补给
- 奖励房间有2个方案：

**[同步块下载失败]**

**升级：武器打印室 → 获得新武器，或“升级”武器**

- 武器打印室会提供3个候选武器，供玩家获取
- 玩家刷新候选项的方式类似哈迪斯
- 刷新次数有2个方案
- 融合打印：将1个奖励列表中的武器战技/技能复制到已有的武器上

**升级：武器强化室 → 强化现有武器**

- 玩家可以从已有武器中选择1个，强化1档品质 
- 普通 → 100%固定数值；100%补正；1个技能槽
- 精良 → 125%固定数值；130%补正；2个技能槽
- 史诗 → 150%固定数值；140%补正；3个技能槽
- 传说 → 175%固定数值；160%补正；4个技能槽

**升级：义体打印室 → 获得新义体，或“融合打印”武器技能**

- 义体打印室会提供3个义体，供玩家获取 
- 待定的随机方案： 
- 方案1：提供完全随机的3个义体
- 方案2：提供同一个部位的3个义体，刷新后部位改变/不变
- 玩家刷新候选项的方式类似哈迪斯（局外升级，有限次数）
- 融合打印：将1个奖励列表中的武器战技/技能复制到已有的武器上

**升级：义体强化室 → 强化现有义体**

- 玩家可以从已有义体中选择1个，强化1档品质

**补给：医疗舱室 → 恢复生命值**

- 恢复一定量生命值

**补给：修复舱室 → 恢复护甲值**

- 恢复一定量护甲值

**补给：弹药仓室 → 强化现有义体**

- 消耗信用点，恢复一定弹药，每种需要单独购买
- 设定上是给公司安保人员的补给房间，花的钱可以报销的，但一定要走一下账冲业绩
- 仅限弹药获取方案1：拾取弹药，需要弹药舱室

**局外：信用授权机 → 获得信用点**

- 获得信用点
- 设定上是借款给公司职员买装备和弹药，你给公司卖命，公司还要你自己花钱补给（笑）

**局外：服务器机房 → 获得「加密图纸」**

- 获得局外升级资源「加密图纸」
- 设定：公司的研发数据机房，因为SDN的存在，2084年的企业数据库几乎不多做加密

**局外：打印研究室 → 获得「打印母料」**

- 获得局外升级资源「打印母料」
- 设定：公司的打印研究室，存储着大量高质量打印母料

**局外内容**

**授权终端**

- 用途： 
- 用于升级公寓、查阅收藏品（也包括解锁的剧情，参考哈迪斯）、获取剧情信息
- 设计： 
- 2084年了，终端应该是所谓的EaaI（everything as an interface，我瞎编的，但这个概念是真的），玩家应该是跑到一个桌子边，手按到桌子上验证SDN访问权限和个人信息权限，然后就能在义眼里看到自己的电脑桌面了（弹UI）。3D资源设计上，桌上应该要有一个可以和手交互、验证授权？ @Arthur Li @Ke Huang

**融合打印舱**

- 用途： 
- 死亡后玩家从这里复活 
- 每次会拥有新的身体、同样的记忆栈
- 没有任何义体
- 玩家可以和打印舱交互，等效于局内的义体打印室，提前购买义体进入战局 
- 玩家只会刷新到带回公寓过的义体
- 默认刷新出的义体品质为普通
- 购买和再次刷新都需要消耗信用点
- Pros：提供独特玩法；提高局内局外资源关联性（信用点）
- Cons：拉长了玩家复活后进入下一局的等待时间（玩家会不停刷），但考虑到我们没有哈迪斯那么丰富的局外剧情资源，也许不是坏事？
- 升级： 
- 玩家可以升级打印舱精度 
- 更高的精度会为玩家提供更高概率，随机出更高品质的义体
- 可以解锁融合打印功能
- 视觉设定： 
- 半床半仓的打印机，参考普罗米修斯里的手术床
- @Arthur Li @Ke Huang
- 故事设定： 
- 因为所有的授权都是数字化打印后即焚的，因此所有的授权都只有一次有效，并且每次需要检索（re-roll）需要重新花费信用点
- 越高级的黑市声望可以让玩家配置更多检索条件，且有更高概率刷出对应条件的装备，玩家在终端升级的就是自己的黑市声望
- 游戏节奏： 
- 玩家在局外就可以用信用点re-roll武器，理论上可以刷一把无敌build直接开打，但因为信用点在局内局外等价，因此其实只是降低了后期玩家的前期战斗难度，作为一种后期游戏提速手段。
- 另外，由于刷出的装备品质和关键字需要用黑市声望2个条件升级，因此还是有很长的局外养成线

**武器打印机**

- 用途： 
- 玩家每次可以在这里领取一把免费武器，重新开始战斗（参考死亡细胞）
- 玩家可以和打印机交互，等效于局内的武器打印机 
- 玩家只会刷新到带回公寓过的武器
- 默认刷新出的义体品质为普通
- 第二次购买和再次刷新需要消耗信用点
- Pros&Cons：same as义体打印机
- 升级： 
- 玩家可以升级打印舱精度 
- 更高的精度会为玩家提供更高概率，随机出更高品质的义体
- 可以解锁融合打印功能
- 设定： 
- 玩家回到公寓会第一时间被扫描记忆栈更新内容，解锁带回来的道具和信息
