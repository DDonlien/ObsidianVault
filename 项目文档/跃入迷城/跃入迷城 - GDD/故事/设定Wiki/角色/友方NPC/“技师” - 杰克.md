
**“技师” - 杰克**

```Jack the Mechanic```
**人物形象**

**参考：**

- 《双城之战》维克托；《2077》老维

**形象：**

- 肤色偏棕红？我想象的是印度人或者中亚人，男性
- 瘦窄的脸，185高的瘦男人，脸刮的很干净，右眼的位置整个已经改造成了特殊的显微镜，延续到耳后有一组特别的三针插线系统，管线连接到不知道哪里。右手也是完全暴露在外的机械化义体，但显然比浩的要更精密，至少看起来还是人类尺寸的手，改造只是为了便于执行手术。

**动作：**

- 类似向前靠撑着柜台台面（因为玩家见他都是在中立的市场楼层）。

**服饰：**

- 浅蓝色的医用翻领T恤，右手的部分破损便于露出义体，衣服上用挂钩/魔术贴挂着一些检测设备的末端，便于随手拿取，衣服似乎还粘着一些血迹








**角色卡**

**第一部分：核心概念**

- **代号 / 姓名:**
- 杰克 (Jack)，人们都叫他“技师”。
- **身份 / 阵营:**
- 独立的义体医生，黑市中最顶尖的改造专家，偶尔会接“野望(Wild Watch)”的单子。
- **核心驱动力:**
- 追求义体改造的艺术性与完美，将每一个作品都视为超越人体局限的艺术品，金钱和阵营对他而言并不重要。
- **功能性质:**
- [✓] **街角观察者:** 在前者基础上，其本体会出现在安全区或已清场的战斗区域。可当面对话、提供情报、任务或交易。*(他有一个隐藏在城市角落的改装店，玩家可以在那里安装独特的、非卖品的义体或升级。)*

**第二部分：角色深度**

- **个性 / 行为风格:**
- 专注、偏执、沉默寡言。只对技术和义体感兴趣，对人际交往显得笨拙。当谈及自己的作品时会变得异常兴奋和健谈。istp
- **与玩家的关系:**
- 工匠与客户 → 实验者与实验品。

**第三部分：叙事弧光**

- **关键过往:**
- 曾是东海工业的首席义体设计师，因其设计的义体被用于残酷的镇压行动而心灰意冷，辞职后开设了自己的地下诊所，只为那些“懂得欣赏艺术”的人服务。
- **初次邂逅:**
- 玩家初次访问商店楼层。
- 可以花钱购买至多3个义体（但不能随机）。
- **最终宿命:**
- 为了完成他心目中最完美的作品（可能是为玩家或他自己安装的某个终极义体），他可能会请求玩家去寻找极其危险的材料，并最终在完成杰作后心满意足地隐退（或死去，并未玩家提供一个特别的义体奖励）。他的摊位会让给本来作为他助手的机器人。
- **对结局的影响:**
- [✓] **部分影响 (Enrichment):** 角色的故事线会“润色”主线结局。不会增加新的结局选项，但会改变结局动画的细节、后日谈的描述、或最终战役的参与人员等。
- *(完成他的任务线能让玩家获得独一无二的强力义体，这可能会让最终的战斗变得更容易，或解锁某些特殊的过场动画。)*
