
**战斗体验Metrics**

**整体**

1. 跃入迷城追求一种：**“以高挑战、重策略的类魂机制为基础，融合了John Wick式流畅、精准、致命的战斗风格，并以Roguelike的强力Build构筑（轮椅流派）作为玩家核心追求和爽点来源”**的战斗体验。
2. 或者简单来说是：**“始于类魂苦战，终于轮椅碾压，全程贯彻John Wick般的紧张与流畅交织感”**。
3. 我们使用1-10分的数值标度，从敌人挑战性和玩家战斗力两个维度设计游戏的战斗体验曲线，并与其他游戏对比，规则如下：
1. **敌人挑战性 (Enemy Challenge):** 1 (几乎无威胁，休闲/教学) - 10 (主流商业游戏中最难的挑战，如顶级Boss/最高难度)。 **只评估敌人本身** (血量、伤害、攻击模式、AI、密度、机制复杂度等)。
2. **玩家战斗力 (Player Power Level):** 1 (几乎无反抗能力，纯粹逃生) - 10 (绝对碾压，如无双类割草)。**只评估玩家角色** (攻击力、技能效果、可用工具、成长潜力、机动性、防御力、Build上限等)。
3. **“游戏难吗？”**主要体现在**敌人挑战性**与**玩家战斗力**的**差值**上，但这不影响两个维度的独立评分。
4. **不同游戏之间的数值是平衡的**：一个开始2分战斗力的黑魂的敌人，应该和一个开始2分战斗力的只狼的敌人，在各自的游戏全盘视角下难度近似，即使很明显他们不会配置相同的机制和数值。
5. **相同游戏内的开始和结束数值，会反应对应的数值和机制强度**：例如一个敌人在开始时，数值是100%，机制是100%，终盘数值是500%，机制是200%，那它开始到结束的威胁应该就是从2分涨到7分（不严谨的距离，因为一般数值会暴涨，而我们会适当压缩打分）。 
1. 这里的关键在于能体现玩家和敌人成长的价值：一个**终盘战斗力7分的玩家应该和终盘挑战性7分的敌人打的难舍难分，与此同时应该碾压开始挑战性2分的敌人**。
6. **锚点固定:** 1分代表体验的下限（几乎无挑战/无能力），10分代表体验的上限（能想象到的主流游戏最高挑战/最强战力）。这两个锚点是**绝对的**，用于校准整个量表。
7. **关注“体感”而非“倍率”:** 评分的依据是玩家在该阶段面对敌人时的**主观感受**，以及角色在该阶段所能发挥的**相对效能**。我们更关注的是“这个敌人比之前的难打**多少**？”或者“我现在比之前强了**多少**？”，而不是“敌人血量翻了**几倍**？”。
8. **边际效用递减式压缩:** 在这个1-10的量表上，越往高端走，每提升1分所代表的**实际数值/机制复杂度**的**绝对增量**可能越大。 
- 例如，敌人挑战性从2提升到3，可能只是敌人多了个简单的冲刺技能，血量提升50%。
- 而敌人挑战性从8提升到9，可能意味着敌人增加了极其复杂的连招、读指令、高额减伤、血量翻倍等等。虽然都是提升1分，但后者代表的实际提升远大于前者。
- 同样，玩家战斗力从3提升到4，可能是拿了把好点的武器；从9提升到10，可能是凑齐了终极套装/Build，获得了指数级的提升。
4. 差值的定义规则如下：

差值衡量的是在某个特定时间点，玩家**主观感受到的的游戏难度或压力程度**。它是基于我们对“敌人挑战性”和“玩家战斗力”两个独立维度的评分计算得出的。

- **正值 (+1 到 +9): 玩家优势区间 - 感觉“简单”或“爽快”**
- **+1 到 +3 (略占优势 / 轻松应对 / Manageable):** 玩家感觉自己的能力稍稍超过当前面对的挑战。战斗需要一定的注意力，但容错率较高，玩家可以比较自信地推进。这通常是标准难度下玩家获得一些关键升级后的体验，或者 ARPG 中装备小有提升后刷怪的感觉。 
- *例子: Hades 终盘 (+1), 咩咩启示录 终盘 (+1)*
- **+4 到 +6 (明显优势 / 割草快感 / Empowered / Easy):** 玩家感觉自己比敌人强大得多。可以轻松消灭大部分敌人，甚至可以进行一些实验性的操作或打法。这接近于传统意义上的“简单”难度，或是玩家构筑的 Build 开始显现威力，体验到力量提升的爽快感。 
- *例子: Diablo 3 开始 (-1) 到 终盘 (+0, 但在低层秘境中这个差值会更大，达到此区间)*
- **+7 到 +9 (绝对优势 / 无双 / God-Mode / Trivial):** 玩家几乎可以无视敌人的威胁，进行单方面的碾压。这代表了极端的 Power Fantasy，通常出现在“剧情模式”、开启作弊或类似《真·三国无双》的游戏体验中。 
- *例子: 吸血鬼幸存者 终盘 (+2，但其割草体验的爽快感接近更高区间)*, *无双类游戏*
- **零 (0): 力量均势区间 - 感觉“平衡”或“刺激”**
- **0 (旗鼓相当 / 针锋相对 / Balanced / Fair):** 玩家的能力与敌人威胁大致相抵。每一次战斗都需要玩家集中精神，合理运用策略和技巧。成功伴随着紧张后的放松，失败则感觉“可以做得更好”。这是很多动作游戏“困难”难度或 Boss 战所追求的平衡点，也是技巧型玩家最喜欢的状态。 
- *例子: **跃入迷城 终盘 (目标体验, 0)**, 火线迈阿密 (0), 恶意不息 终盘 (-1, 接近0)*
- **负值 (-1 到 -9): 玩家劣势区间 - 感觉“困难”或“受苦”**
- **-1 到 -3 (稍处劣势 / 充满挑战 / Challenging / Hard):** 玩家感觉敌人的威胁略大于自己的应对能力。需要步步为营，精打细算，充分利用所有可用资源和机制。失误的代价开始变高。这是典型的“困难”难度体验，或是魂类游戏中后期、ARPG 开荒期的感受。 
- *例子: 黑暗之魂 1/2/3/Bloodborne 终盘 (-1), 死亡细胞 终盘 (-1), 艾尔登法环 终盘 (-1)*
- **-4 到 -6 (明显劣势 / 步履维艰 / Punishing / Very Hard):** 玩家感觉自己显著弱于敌人。生存压力巨大，推进缓慢，需要反复尝试和学习。每一次成功都来之不易。这通常是魂类游戏的开局体验，或是极高难度设定下的感受。 
- *例子: **跃入迷城 开始 (-4)**, 黑暗之魂 1/2/3/Bloodborne/恶意不息 开始 (-3 到 -4), Sekiro 开始 (-3, 但体感更难), 逃离鸭科夫 终盘 (-1, 但过程压力极大，接近此区间)*
- **-7 到 -9 (绝对劣势 / 几无胜算 / Brutal / Near Impossible):** 玩家感觉几乎无法对抗当前的挑战。可能需要依赖特定的策略、运气、或者这本身就是一种叙事或机制设计（例如某些生存恐怖游戏的开局）。失败是常态，生存本身就是胜利。 
- *例子: 逃离鸭科夫 开始 (-7), 流放之路/PoE2 开始 (-5), 以撒结合 开局脸黑 (-3 到 -5)*
5. 跃入迷城追求的体验如下：

**基于新的开盘设定（敌人挑战性7，玩家战斗力4），我们重新梳理《跃入迷城》从开盘到终盘的整体体验：**

1. **开盘体验 (Start: 敌人挑战性 7 vs 玩家战斗力 4 = 难度体验 -3):**
- **核心感受：压力下的探索与适应。** 玩家进入不夜城时，立刻会感受到这是一个危险的世界。敌人（7分挑战性）无论是基础数值还是攻击模式/AI，都构成了显著威胁，需要玩家认真对待，符合“魂Like”的基调。然而，玩家并非手无寸铁（4分战斗力），拥有比系列早期魂作更灵活或更多样化的基础工具（可能是更流畅的移动/翻滚、不错的初始武器性能、基础的近战+远程切换能力）。
- **玩家行为驱动：** 这个-3的难度差促使玩家必须**快速学习**游戏的核心机制：观察敌人、管理资源（体力/弹药）、掌握闪避时机、并开始尝试结合近战与远程的“John Wick”式战斗风格。玩家会感到挑战，但每一次成功清空房间或击败一个小型精英敌人，都会带来“我能行”的正反馈，驱动玩家探索和成长。相比-4的体验，这里的挫败感可能稍低，玩家能更快地找到“反击”的感觉。
- **关键学习点：** 混合战斗的初步运用、资源控制、敌人行为模式识别、以及适应俯视角下的空间感和战斗节奏。
2. **成长过程 (Mid-game Progression):**
- **能力逐步解锁与Build萌芽：** 随着玩家在一次次轮回中获取新的武器、义体、技能模块，并可能进行局外养成强化，“玩家战斗力”评分会稳步提升（从4向10迈进）。玩家会明显感觉到自己的选择变多，战斗方式开始多样化，一些基础的Build协同效果开始显现（例如，某个义体强化了特定类型的武器，或某个技能组合产生了不错的连携）。
- **挑战升级与遭遇多样化：** 同时，更深层的关卡会引入更强大的敌人（挑战性从7向9提升）。这些敌人不仅数值更高，还会拥有更复杂的攻击模式、更狡猾的AI行为（例如团队配合、利用环境）、以及更棘手的精英词缀或组合。关卡设计本身也可能增加复杂度（例如更多的环境互动、陷阱、特殊的遭遇战设计）。
- **体验波动与策略深化：** 这个阶段玩家的“难度体验差”是**抛物线**动态变化的。获得强力组件后可能会暂时觉得轻松（差值接近0或为正），但很快又会遇到新的、更强的挑战，再次感到压力（差值可能回到-2或-3）。玩家需要不断根据获得的资源调整自己的Build方向，并深化对战斗系统和敌人机制的理解，策略性变得越来越重要。
3. **终盘体验 (Endgame: 敌人挑战性 9 vs 玩家战斗力 10 = 难度体验 +1):**
- **核心感受：巅峰对决与掌控下的华丽战斗。** 当玩家通过精心构筑，达到了游戏系统内设定的最强战斗力水平（10分），并面对游戏中最顶级的挑战（9分敌人，例如最终Boss或隐藏强敌）时，体验将升华。
- **微妙优势与自信执行：** 玩家明确知道自己拥有当前版本的最优解（“轮椅”成型），面对极度危险的敌人时，能够感受到一种**细微但确定的优势**（+1）。这种优势让玩家敢于更主动地进攻，更自信地执行John Wick式的流畅、精准、甚至带有表演性质的战斗操作。
- **高压下的“爽点”：** 战斗依然极其紧张刺激，因为敌人（9分挑战性）的攻击依然致命，机制依然复杂。但玩家强大的能力（10分战斗力）使得容错率略有提高，并且能够更有效地将策略转化为胜势。这种在高压下依然能“秀”起来，并最终掌控战局的感觉，正是“轮椅流派”在高难度游戏中的核心爽点——**并非无敌，而是以绝对实力凌驾于顶级挑战之上**。
- **成长闭环与重玩价值：** 达成这种终盘体验是对玩家整个游戏理解和Build构筑能力的终极肯定，提供了极强的满足感。同时，由于Build的多样性，玩家可以通过尝试不同的“轮椅”构筑来重复体验这种巅峰对决，增加了游戏的重玩价值。
