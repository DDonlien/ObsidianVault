
**“支柱” - 李诺德**

```**NarodLee the Pillar**```


```**Boss - 3**```


```**阵营设定**```
**社会身份：**

- 中间人

**关卡视觉风格：**

- Boss关：只有中央电梯厅的楼层，可移动面积比其他楼层小非常多（差不多是8个电梯厅的面积围绕电梯厅排布），完全没有家具，并且没有阳台栏杆，开放边缘（掉落致死）

```**战斗设定**```
**准备阶段：**

- 完全无准备的突然袭击，可能是巨拳或者机枪扫射，然后就爬到窗边正式开战，可能从任何一个角度开始。

**战斗阶段1：**

- 巨大的机器人从楼外攻击玩家，时不时会换角度，一次只能攻击一个扇区
- 近战攻击：扇形挥拳
- 远程攻击：导弹弹幕或者机枪扫射

**战斗阶段2：**

- 机器人护甲值降低到1/2时进入2阶段。2阶段机器人的操作员会跳出来，协同作战
- 机器人
- 和1阶段一样
- 驾驶员
- 手持步枪，同时有手雷、地雷、小刀，会执行各类战术操作
