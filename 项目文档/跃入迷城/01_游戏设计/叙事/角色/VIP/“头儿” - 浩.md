
**“头儿” - 浩**

```**Hall the Boss**```


```**Boss - 1** ```
**人设**

**形象**

- 显然超过常规程度的健壮男性，身高2.2m，体重300kg，蒙古人，有蒙古将军一样的大力士身材
- 身上装备着大量义体，但基本属于工业机械风格，追求力大砖飞和高性价比，拥有额外2只（共四只）机械臂。显然财力不足以支撑他购买高端的仿生义体
- 粗糙的皮肤上有些伤疤，没有进行融合打印修复也说明了他的社会地位并不真的很高

**背景**

- Hall曾是一名普通的工头，在一次事故中高位截瘫。他耗尽所有资源，购得了一枚非法的记忆栈，才得以恢复行动能力。
- 他是劳旅体系的坚定支持者，同时也深谙体系的冷酷。他在东区不仅负责劳工调度，还在更深层的交易中掌握主动，确保“资源”在劳旅与其他势力之间高效流转。关于这些“资源”的具体细节，工人们都心知肚明，却从不敢提起。
- 对Hall而言，权力就是一切。他明白自己的身体已经走向衰败，但他坚信，只要劳旅体系存在，他的地位就不会动摇。

**社会身份**

- 独立人力外包公司“劳旅”（名待定）在不夜城东区的总工头。他在劳旅中位高权重，直接统辖东区的所有劳工事务。从工地调度到危机处理，他是劳旅这一民间巨兽在东区的代言人。他的权力并非来自中立公司的支持，而是凭借劳旅对不夜城底层资源的绝对掌控，与中立公司保持微妙而务实的合作关系。

**社会定位**

- [劳旅 (Labor)](../../阵营/民间组织/劳旅.md)的@不夜城地区主管。
- 在普通工人眼中，他是劳旅体系冷酷无情的代表。他压榨劳工，却也能在某些情况下为工人争取利益。这种复杂的形象使他成为一种令人敬畏的存在。
- 对特许政府和中立公司来说，**浩**是劳旅体系的一部分，必要时可以合作，但绝不可完全信任。他对资源的掌控能力令人印象深刻，但在更大的利益面前，他的影响力仍然局限在东区范围内。
- **浩**清楚地认识到自己是劳旅的一颗重要棋子，但他也知道，没有他，这盘棋会走得更艰难。他不求超越劳旅，但绝不会容忍自己的地位被动摇。

**第一阶段**

触发条件：无

特殊机制：无，场地内本来应该就有一批小弟

**攻击机制**

1. **Attack 1：**普通近战攻击
2. **Spell 1：**霰弹枪普通攻击
1. 抬手动作：霰弹枪瞄准
2. 瞄准玩家，{0.5s}后转身速率降低追踪玩家
3. 然后向前面朝方向步行，并且每隔{0.5s}发射一次霰弹枪
4. 可以用跟踪射击实现
3. **Spell 2：**机枪扫射
1. 抬手动作：机枪瞄准玩家
2. 引导后开始机枪扫射
3. 期间转身速率降低，但持续尝试追踪玩家，持续{3s}
4. 可以用跟踪射击实现
4. **Spell 3：**霰弹枪突击
1. 抬手动作：切换到霰弹枪、换弹
2. 向玩家方向快速冲刺
3. 停止追踪，霰弹枪攻击面前
5. **Spell 4：**格挡抵近
1. 抬手动作：像拳击手一样把机械臂挡在身前
2. 向玩家方向移动，期间正面的伤害免疫
3. 如果在持续时间内靠近了玩家，进行一次顺劈斩，造成周身范围伤害 + 击退

**辅助机制**

1. 看着玩家等待
2. 看着玩家移动
