```**设计指引：** 用于快速构建角色核心，并通过叙事碎片呈现给玩家。
```
**第一部分：核心概念**

- **代号 / 姓名:** *(一个人的代号往往比真名更能揭示其本质。)*
- **身份 / 阵营:** *(角色的社会定位与归属，公开或隐藏。例如：财团执行官、情报掮客、流浪AI、拾荒者。)*
- **核心驱动力:** *(角色行动的根本目标或执念。例如：复仇、寻找真相、求生、守护秘密。)*
- **功能性质:** *(选择1项，决定玩家与角色的互动层级与叙事方式。)*
- **文本幽灵 (Text Ghost):** 纯粹的背景叙事者。仅通过数据碎片、环境日志、物品描述、尸体留言等静态文本存在，玩家永远无法直接互动。
- **通讯魅影 (Holo-Caller):** 在**文本幽灵**的基础上，会通过全息通讯、加密留言等远程、非实体的方式与玩家接触。是重要的情报源或任务发布者，但其真实身份和位置成谜。
- **街角观察者 (Street Observer):** 在**通讯魅影**的基础上，其本体会出现在安全区或已清场的战斗区域。可当面对话、提供情报、任务或交易。可被玩家攻击/击杀，但不会主动介入战斗。*(参考：《艾尔登法环》的流浪商人、《哈迪斯》的卡戎)*
- **战场过客 (Combat Zone Bystander):** 在**街角观察者**的基础上，会出现在仍有敌人的危险区域，但因其特殊立场或能力而不会主动参与战斗。可能是任务目标（救援/护送），也可能只是一个让你感到诡异的旁观者。*(参考：《哈迪斯2》的赫拉克勒斯、涅墨西斯)*
- **临时盟友 (Temporary Ally):** 在**街角观察者**的基础上，在特定区域、任务或条件下，会受邀与玩家并肩作战。合作关系可能是脆弱和短暂的，为玩家提供暂时性的战力。*(参考：《艾尔登法环》中可召唤参与BOSS战的NPC)*
- **敌人 (Enemy):** 玩家需要面对的对手。其故事通过战斗前后的对话、技能招式、掉落物描述等方式呈现。战斗后，其状态可能转变为上述任一非敌对状态（如被击败后成为“街角观察者”），或彻底死亡，或复活（BOSS或者其他有记忆栈的角色）。

**第二部分：角色深度**

- **个性 / 行为风格:** *(角色的言行举止，以及对玩家的基本态度。)*
- **与玩家的关系:** *(初始关系与潜在的动态转变。例如：引导者 → 背叛者，竞争者 → 盟友。)*

**第三部分：叙事弧光**

- **关键过往:** *(游戏开始前，决定了TA如今状态的核心事件。)*
- **初次邂逅:** *(玩家首次接触该角色的场景，以及留下的核心谜团。)*
- **最终宿命:** *(角色故事的终局，是否由玩家选择决定。例如：达成目标后消逝、疯狂、成为新BOSS。)*
- **对结局的影响:**
- [ ] **无影响 (Flavor):** 角色的故事线完全独立，用于丰富世界观或提供奖励，但无论是否完成，都不会对主线结局本身产生任何改变。
- *(范例：《艾尔登法环》的战士壶亚历山大，他的故事感人至深，但与任何结局的触发都无关。)*
- [ ] **部分影响 (Enrichment):** 角色的故事线会“润色”主线结局。不会增加新的结局选项，但会改变结局动画的细节、后日谈的描述、或最终战役的参与人员等。
- *(范例：《质量效应2》的队友忠诚任务，完成与否决定了队友能否在最终任务中存活，从而影响结局动画和《质量效应3》的开局。)*
- [ ] **主动影响 (Optional Ending):** 角色的故事线是解锁一个**全新可选结局**的钥匙。玩家可以遵循默认主线，也可以通过完成该角色的任务来开启一个完全不同的结局分支。
- *(范例：《艾尔登法环》的菈妮，完成她的漫长任务线后，才能在最终选择时触发“星星时代”结局。)*
- [ ] **关键影响 (Gatekeeper):** 该角色是主线剧情的“守门人”。玩家必须推进其故事线的关键节点，才能继续主线任务或进入最终结局。
- *(范例：《艾尔登法环》的梅琳娜，玩家必须与她合作，才能前往王城并最终烧毁黄金树，这是通往任何结局的必经之路。)*
- [ ] **必然选项 (The Main Story):** 该角色的个人故事**就是**主线故事。玩家的旅程与该角色的命运深度绑定，无法分割。推进主线即是体验该角色的故事。
- *(范例：《生化奇兵：无限》的伊丽莎白。玩家（作为布克）的核心任务就是“找到她，带走她”。伊丽莎白的身世、她的特殊能力以及她与世界的真相构成了游戏的绝对核心，整个故事因她而展开，玩家的旅程完全由她的存在而定义。)*

2. 死亡搁浅 - 希格斯·蒙纳汉

**[希格斯]**

```**设计指引：** 用于快速构建角色核心，并通过叙事碎片呈现给玩家。
```

**第一部分：核心概念**

- **代号 / 姓名:***希格斯·蒙纳汉 (Higgs Monaghan)，自称“上帝粒子”。*
- **身份 / 阵营:***分裂主义军事组织“人类末日”的领袖，遵从灭绝体意志的遣返者。*
- **核心驱动力:***阻止UCA的重建，并加速最终的“死亡搁浅”大灭绝事件，将世界彻底推向终结。*
- **功能性质:**
- [✓] **敌人:** 会与玩家直接为敌的对手。其故事通过战斗风格、技能招式、战前战后对话及掉落物来呈现。战后，其状态可能转变为非敌对状态，或彻底死亡。\*

**第二部分：角色深度**

- **个性 / 行为风格:***浮夸、自恋、戏剧化，极度沉迷于自己的力量。渴望他人的关注（点赞），言行充满对主角山姆的挑衅和扭曲的游戏心态。*
- **与玩家的关系:***宿敌。*

**第三部分：叙事弧光**

- **关键过往:***曾是布里吉斯的一名普通搬运工，因其DOOMS体质被灭绝体艾米莉选中并赋予力量，从此成为她忠实的末日代理人。*
- **初次邂逅:***在游戏初期，他便通过全息通讯不断骚扰和威胁山姆，并在焚化场外首次与山姆正面冲突。*
- **最终宿命:***在冥滩上与山姆进行一系列决斗后被彻底击败，力量被剥夺。最终由玩家决定其生死（或将其永远遗弃在冥滩）。*
- **对结局的影响:**
- [✓] **关键影响 (Gatekeeper):** 该角色是主线剧情的“守门人”。玩家必须推进其故事线的关键节点，才能继续主线任务或进入最终结局。 
- *(作为贯穿前中期的主要反派，玩家必须击败他并瓦解他的阻挠，才能将开罗尔网络连接至西海岸，从而触发游戏的最终章。他是通往结局的路上必须推倒的墙。)*
