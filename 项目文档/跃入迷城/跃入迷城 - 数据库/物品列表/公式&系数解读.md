
**公式&系数解读**

**伤害系数**

= T3 \* 1.2 / BQ3 / BR3  \* ( 0.8 + 0.4 / (1 + EXP(2 \* (T3 - 0.5))))

1. **1.2 —— 全局 DPS 修正系数 (The Global Scaler)**

出现在：T21 \* 1.2 / ...

- **作用：** 这是一个**全局的强度开关**。
- **解读：** 如果没有这个系数，你的公式本质上是 时间 / 弹片数。加入 1.2 意味着你人为设定了 **“标准单位时间内的期望伤害倍率”**。
- **调整策略：**
- 如果你觉得游戏里**所有武器**杀怪都太慢了，就把 1.2 改成 1.5。
- 如果你觉得所有武器都太超模了，就改成 1.0。
- 它不影响平衡性（快慢武器之间的差距），它只影响整体游戏的 TTK（击杀所需时间）。
2. **0.8 和 0.4 —— 风险补偿区间 (The Risk Bonus Range)**

出现在：( 0.8 + 0.4 / ... )

这两个数定义了\*\*“最快武器”**和**“最慢武器”**之间的**额外DPS收益差\*\*。

- **0.8 (基准线 / 地板)**：
- 这是给予极快射速武器（低风险、低硬直）的**基础效率**。
- 也就是说，极快武器的伤害效率是标准值的 80%。为什么要打折？因为它们容错率高，空一枪不心疼，走位灵活。
- **0.4 (最大红利 / 奖金池)**：
- 这是给予极慢射速武器（高风险、高硬直）的**最大额外补偿**。
- **计算逻辑：** 整个 Sigmoid 函数的取值范围是 0~1。
- 最快武器（Sigmoid ≈ 0）：系数 = $0.8 + 0 = 0.8$。
- 最慢武器（Sigmoid ≈ 1）：系数 = $0.8 + 0.4 = 1.2$。
- **结论：** 这个 0.4 决定了，一把超级慢的重武器，其 **DPS（不是单发伤害，是DPS）** 上限会比一把超级快的小手枪高出 **50%** ($1.2 / 0.8 = 1.5$)。这是为了奖励玩家承担“站桩输出”或“前摇易被打断”的风险。
3. **0.75 —— 风险阈值中点 (The Pivot Point)**

出现在：(T21 - 0.75)

- **作用：** 定义了什么叫 **“不快也不慢的标准武器”**。
- **解读：**
- 当武器攻击时长 $T = 0.75$ 秒时，Sigmoid 函数正好处于中间点（0.5）。
- 此时的效率系数是 $0.8 + 0.2 = 1.0$。
- 也就是说，**0.75秒**是你设计的一把“标准步枪”或“标准大剑”的攻击节奏。比这个慢的武器开始享受正向收益（DPS > 1.0），比这个快的武器受到负向修正（DPS < 1.0）。
4. -2 —— 敏感度曲线 (The Slope)

出现在：EXP(-2 \* ...)

- **作用：** 决定了收益变化的**陡峭程度**。
- **解读：**
- 如果这个数字很大（比如 -10），那么只要时间稍微超过 0.75秒，奖励就会瞬间拉满，变成一个阶梯跳跃。
- 现在的 **-2** 是一个相对平滑的曲线。它意味着从 0.5秒 到 1.0秒 的变化过程中，DPS的提升是线性偏软的，鼓励玩家逐渐适应慢速武器，而不是强迫玩家必须用极慢的武器。
