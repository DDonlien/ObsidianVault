# 控制
同步回合制，每次玩家释放指令，敌人也释放指令
玩家在六边形格子上移动，以下动作结算1个回合：
1. 移动；一般情况下只可以往面前的格子移动1格
2. 转身；一般情况下每次转1条边（60°）
3. 运气；加载一条法术链，一条法术链的定义详见[[行动块]]部分
4. 发功；释放所有已经加载的法术链，按顺序释放，释放范围和方向取决于各个法术链本身
5. 等待；等待1回合
玩家在1格回合内还可以额外做一些不受限制的行为：
1. 调整数个已加载的法术链先后顺序
2. 调整当前还未加载的法术链中，法术行动块的顺序
3. 移除已加载的法术链
4. 使用道具
当玩家或敌人结算回合行动时，总是先移动、后施法
不同于玩家，当决定攻击玩家时，敌人会额外加载一个“攻击”回合，并在下一个回合释放所有已加载的法术。敌人的“攻击”回合结算晚于玩家的施法回合，因此整个游戏的单回合结算顺序如下：
- 移动/旋转 -> 施法(玩家) -> 攻击(敌人)

### 属性
生命：玩家可以拥有阴血和阳血，在游戏中通过特殊手段（角色、奖励）切换，默认为阳血

属性克制：相克的属性
