### 组合
**自增**  
[[金]] + [[金]] = [[金]] · {Z}  
**相生**  
[[金]] + [[水]] = [[汞]] · {Z}  
**相克**  
[[金]] + [[土]] = [[无]] · {X}  
**次生**  
[[金]] + [[木]] = [[锋]] · {Y}
### 说明
组合符号为“+”，代表顺序不重要
组合符号为“>”，代表顺序重要

符号“·”用于描述行动块等级

N = 当前行动块的等级
X = 组合行动块中等级较低者
Y = 组合行动块中等级较高者
Z = 组合行动块的等级合
