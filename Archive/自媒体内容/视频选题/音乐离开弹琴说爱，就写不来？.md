# 讲稿

前两天我和我女友聊天，提到最近我音乐创作有点瓶颈，感觉自己生活很稳定，稍微有点找不到创作主题，好像音乐离开了狗血的爱恨情仇和年轻人的一腔怒火，就很难展开创作，不像做游戏、写小说，什么主题开展下去的都非常多。结果她她狠狠吐槽了我一顿，并附上了某乎名言“你能不能先确认是不是，再解释为什么啊？”

这到给了我一个视频灵感：是不是在今天音乐创作和音乐的受众，相比其他8种艺术形式，尤其喜欢爱恨主题？

于是我打算用可以访问的信息做一个简单的统计给大家分享。

开始研究之前我们要先界定一些边界条件。首先9大艺术分别是
# 选题
横评现代9大艺术的主题内容，评估是否音乐相比其他类型尤其喜欢选用爱情（爱或恨皆可）作为创作主题，以2024年的创作内容为取样。

# 取样
1. 只关注销量/播放量，展示消费者喜好
2. 综合各表单前10名，合并同类项
3. 刨除了绘画、雕塑、建筑、舞蹈、戏剧类型，实在不懂
