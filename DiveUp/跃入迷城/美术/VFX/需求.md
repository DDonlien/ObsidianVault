# 技能

## 瞄准VFX
技能的VFX提醒应该有3个步骤组成：
1. UI层面的瞄准，在释放前使用，半透明，仅玩家自己可见
2. 释放后的VFX
3. Loading期间的VFX
- [ ] 指向单点技能
	- 不涉及长度
	- 不涉及蓄力
- [ ] 非指向线性技能
	- 长度自适应
	- 宽度自适应
	- 兼容蓄力
	- 兼容无限/超长距离情况（仅提示方向、不提示距离）
- [ ] 非指向扇形技能
	- 长度自适应
	- 兼容蓄力
	- 宽度自适应
- [ ] 非指向选点技能
	- 长度自适应
	- 宽度自适应
	- 兼容蓄力
